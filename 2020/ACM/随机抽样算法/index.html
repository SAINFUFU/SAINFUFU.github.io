随机抽样算法

2020-05-19
ACM-随机化算法

题目描述

设计一个算法从n个记录中选取m个记录

分析

当从n个记录中选取m个时，根据概率论中的抽签模型，每个记录被抽到的概率是一样的，都为m/n。因此根据这个原理编写算法如下：
依次对n个记录进行判断，如果当前随机产生的值小于当前概率，则选取这条记录。
时间复杂度分析：O(n)

算法描述：void select(int n,int m,int s[])
输入：记录总数n，选取数m，选择记录数组s[]
输出：选择记录数组s[]
过程：
1.遍历n个记录，同时记录已经选取的记录数x
2.随机生成一个数r在0-1之间
3.if(r<(m-x)/(n-i-1)) （即如果u<当前该记录被选择的概率），就选择该记录
4.否则，遍历下一条记录。
5.当m个记录都被选择之后，就直接退出循环

代码

void select(int n,int m,int s[])
{
    RandomNumber rnd;
    int x = 0;//记录已经被选择的个数
    for(int i = 0;i<n&&x<m;i++)//当m个记录选择完时直接退出循环
    {
      double r = rnd.fRandom();//随机生成0-1之间的数
      if(r*(n-i-1)<(m-x))//改用乘法更便于判断
      {
          x++;//被选择的数+1
          s[i] = 1;//第i个被选择
      }
    }
}
