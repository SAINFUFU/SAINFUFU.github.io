运动员最佳配对问题

2020-05-08
ACM-分支限界法

题目描述

羽毛球队有男女运动员各n人，给定2个n×n矩阵P和Q。P[i][j]是男运动员i和女运动员j配对组成混合双打的男运动员竞赛优势；Q[i][j]则是女运动员i和男运动员j配合的女运动员竞赛优势。

由于技术配合和心理状态等各种因素的影响，P[i][j]不一定等于Q[j][i]。男运动员i和女运动员j配对组成混合双打的男女双方竞赛优势为P[i][j]*Q[j][i]。设计一个算法，计算男女运动员的最佳配对法，使各组男女双方竞赛优势的总和达到最大。

分析

因为是配对问题，所以可以将女运动员的顺序固定为序号升序排列（即0,1,2……，n-1）,只对男运动员的顺序进行广度优先搜索。
建立一个节点Node存放当前配对的上界rcost，当前男运动员顺序x[],当前节点的层数s，当前优势cc。
每次固定一个顺序时，将节点的上界rcost更新为原来的rcost-MaxOut[i]（男运动员i的最大优势）+val[i][s]（男运动员i的实际优势）。

对于剪枝函数可以将Node节点的优先级按rcost降序排列。
因此当取出的节点t的上界rcost<=bestc时，可以直接剪枝，因为之后的配对顺序的最大优势也肯定比bestc小，不必再去搜索后面的节点。

另外建立Contest类，用于进行搜索运算，存放目前最大的优势bestc，每个男运动员的最大优势MaxOut[]，最大优势总和（最大上界）MaxSum。

时间复杂度分析：
因为是枚举所有的排列，而限界函数对复杂度没有影响，因此时间复杂度为O(n!)。

代码

#include <iostream>
#include <cstdio>
#include <string.h>
#include <algorithm>
#include <queue>
#include <time.h>

using namespace std;
const int N = 20;
int n;
int p[N][N],q[N][N];
int MaxSum;
class Node//节点
{
    public:
        void Init();
        int x[N],rcost,s,cc;
};
void Node::Init()//节点初始化
{
    for(int i = 0;i<n;i++)
        x[i] = i;
    s = 0;
    cc = 0;
    rcost = MaxSum;
}
bool operator <(Node a,Node b)//重载小于号，定义优先级
{
    return a.rcost<b.rcost;
}
class Contest{
    public:
        void Init();
        int GetResult();
        int bestc;//最优解
        int val[N][N],MaxOut[N];//男运动员i的最大优势
};

void Contest::Init()//初始化
{
    bestc = 0;//初始化最优解
    MaxSum = 0;//初始化最大优势总和
    memset(MaxOut,0,sizeof(MaxOut));
    for(int i = 0;i<n;i++)//i为男运动员编号
    {
        for(int j = 0;j<n;j++)//j为女运动员编号
        {
            val[i][j] = p[i][j]*q[j][i];
            if(val[i][j]>MaxOut[i])
                MaxOut[i] = val[i][j];
        }
        MaxSum += MaxOut[i];
    }

}

int Contest::GetResult()
{
    Init();//初始化
    priority_queue<Node> que;
    Node t;
    t.Init();
    que.push(t);
    while(!que.empty())//队列不空
    {
        t = que.top();//取一节点
        que.pop();
        //如果当前排列的上界<当前最优解，就退出循环
        if(t.s>n-1 || t.rcost<=bestc) break;
        if(t.s == n-1)//如果排列完
        {
            if(val[t.x[n-1]][t.s]+t.cc>bestc)//当前优势>当前最优解
            {
                bestc = val[t.x[n-1]][t.s]+t.cc;//更新最优解
            }
        }
        else//没有排列完
        {
            for(int i = t.s;i<n;i++)//从未排列的男运动员中挑一人
            {
                int cc = t.cc+val[t.x[i]][t.s];
                int rcost = t.rcost - MaxOut[t.x[i]] + val[t.x[i]][t.s];//更新当前排列的上界
                if(rcost>bestc)//如果>当前最优解，就进行扩展
                {
                    Node cur;//新建节点
                    for(int j = 0;j<n;j++)
                        cur.x[j] = t.x[j];
                    cur.x[t.s] = t.x[i];
                    cur.x[i] = t.x[t.s];
                    cur.s = t.s+1;
                    cur.cc = cc;
                    cur.rcost = rcost;
                    que.push(cur);//插入节点
                }
            }
        }
    }
    return bestc;//返回最优解
}

int main()
{
    freopen("input.txt","r",stdin);
    freopen("output.txt","w",stdout);
    clock_t start,end;
    while(cin>>n && n)
    {
        for(int i  = 0;i<n;i++)
            for(int j = 0;j<n;j++)
                cin>>p[i][j];
        for(int i  = 0;i<n;i++)
            for(int j = 0;j<n;j++)
                cin>>q[i][j];
        Contest c;
        start = clock();     
        printf("%d\n",c.GetResult());
        end = clock();
        printf("n = %d,time = %dms\n\n",n,end-start);   
    }
    return 0;
}
